# LibraryManagementSpringBoot

Spring Boot CRUD application for managing library books.

## Run

```bash
mvn spring-boot:run
```

## REST API

- `POST /books`
- `GET /books`

## H2 Console

- URL: `http://localhost:8080/h2-console`
- JDBC URL: `jdbc:h2:mem:librarydb`
- Username: `sa`
- Password: leave blank
