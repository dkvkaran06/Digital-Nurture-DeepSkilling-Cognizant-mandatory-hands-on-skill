## 🚀 How to Run

1. Navigate to the desired project folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open `http://localhost:3000` (React) or `http://localhost:4200` (Angular) in your browser.
