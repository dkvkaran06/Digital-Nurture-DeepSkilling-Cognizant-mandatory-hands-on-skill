# 📘 Week 1 — PL/SQL

## 📌 Overview

This week focuses on **PL/SQL (Procedural Language/SQL)** for Oracle databases. The exercises cover database schema design, stored procedures, functions, triggers, cursors, packages, and error handling for a **Banking System** scenario.

## 📂 Folder Structure

```
Week 1 PL-SQL/
├── Createtables.sql       → Schema creation (Customers, Accounts, Transactions, Loans, Employees)
├── Insertsampledata.sql   → Sample data insertion
├── Excercise1.sql         → Control Structures
├── Excercise2.sql         → Error Handling
├── Excercise3.sql         → Stored Procedures
├── Excercise4.sql         → Functions
├── Excercise5.sql         → Triggers
├── Excercise6.sql         → Cursors
└── Excercise7.sql         → Packages
```

## 🛠️ Tech Stack

- **Language:** PL/SQL
- **Database:** Oracle
- **Concepts:** DDL, DML, Stored Procedures, Functions, Triggers, Cursors, Packages, Exception Handling

## 🚀 How to Run

1. Open **Oracle SQL Developer** or **SQL*Plus**.
2. Run `Createtables.sql` first to create the schema.
3. Run `Insertsampledata.sql` to populate the tables.
4. Execute each exercise file (`Excercise1.sql` through `Excercise7.sql`) sequentially.
